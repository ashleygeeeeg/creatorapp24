# AppCreator24 / AppMaker24 — CreatorApp24

Link **[AppCreator24](https://www.appcreator24.com)** as the Android shell; host the web app from **creatorapp24** on Vercel.

## Deploy

1. Vercel → import **creatorapp24**, root **`frontend`**
2. `REACT_APP_BACKEND_URL` = your API HTTPS origin
3. `GET /api/integrations/appmaker24` for WebView menu URLs
4. Publish on AppCreator24 → set `APPCREATOR24_APP_URL` / `REACT_APP_APPCREATOR24_APP_URL`

See full checklist in this file on Clone or expand env in `.env.example`.
